# 老婆点菜

贾维斯智能点菜系统 — 记住老婆的口味，越用越懂她。

## 功能

- **今日推荐**：基于老婆的口味偏好，每天智能推荐
- **场景点菜**：说"想吃辣的""想吃肉""来点清爽的"，自动匹配
- **全部浏览**：按难度、时间、荤素筛选，搜索菜名/食材
- **口味画像**：记录辣度、咸甜酸偏好、爱吃的菜系、不吃的食材
- **收藏 & 踩**：喜欢就收藏，不喜欢就标记，推荐越用越准
- **点菜历史**：记录每天吃什么，自动学习偏好

## 快速启动

```bash
pip install flask flask-cors
python app.py
```

打开手机浏览器访问 `http://[IP]:8090`

## 技术栈

Flask + 原生 JS + 175 道菜谱 + 个性化推荐引擎
